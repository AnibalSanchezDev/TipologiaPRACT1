## Archivos
* `src/main.py`: Script principal de extraccion de datos con selenium.
* `data/data_[busqueda]_[fecha].csv`: Dataset bruto con las ofertas.
* `requirements.txt`: Archivo de dependencias.

## Cómo usar el código

### Instalación del entorno
```bash
# Crear entorno virtual
python -m venv venv

# Activar entorno
# En Windows:
.\venv\Scripts\activate

# Instalar dependencias
pip install -r requirements.txt
```

### Ejecución programa

python source/main.py [parametro]

En caso de ningun parametro, por default se escribira 'Python'

#### Ejemplos
* python .\source\main.py "Data Scientist"
* python source/main.py Java
* python source/main.py
